# Retail-Sales

## About the Project

This project focuses on analyzing retail sales data using SQL to gain meaningful business insights. The dataset contains transactional sales records, which are queried to understand customer behavior, sales trends, product performance, and revenue distribution.

The project demonstrates practical usage of SQL concepts such as filtering, aggregation, grouping, joins, and subqueries to solve real-world business questions. It is designed to strengthen data analysis skills and showcase the ability to work with structured datasets in a database environment.

## Tech Stack
- SQL
- Relational Database
- Retail Sales Dataset
